# Introducción a Terraform

Terraform es una herramienta de infraestructura como código (IaC) que permite declarar recursos de nube en archivos de configuración...

## ¿Por qué IaC?

La Infraestructura como Código permite:
- Versionado de infraestructura.
- Despliegues repetibles.
- Automatización total.
